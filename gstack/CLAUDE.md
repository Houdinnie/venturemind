## VentureMind Context

### Domain Model

- **10 Swarms**: Engineering, Legal, Financial, Capital, Growth, Web3, Wealth, Mobility, Journey, Central
- **Central Swarm Lead**: Orchestrator, Strategy Agent, Operator Agent, Talent Agent
- **Paperclip Integration**: Company package at `paperclip/venturemind/`
- **SafetyNet**: Non-negotiable HITL triggers, Watchdog Agent, zero-knowledge vault
- **Ingestion**: 7-stage adaptive form, Strategist Agent, 4 parallel verification loops

### Tech Stack

- **Frontend**: React 19 + Vite + Tailwind + shadcn/ui
- **Backend**: FastAPI + LangGraph + PostgreSQL + Redis + Qdrant
- **AI**: Claude via Anthropic API + Groq + OpenAI
- **Agents**: Paperclip (company package) + gstack (engineering workflows)
- **Security**: Zero-knowledge vault, TEE enclaves, AES-256-GCM, Argon2id

### Risk Levels (SafetyNet)

| Level | Threshold | Action |
|-------|-----------|--------|
| AMBER | > $0 | Log all actions, append disclaimers |
| YELLOW | > $500 | Enhanced monitoring, explicit acknowledgement |
| ORANGE | > $2,000 | Explicit multi-sig required |
| RED | > $10,000 | Full multi-sig + 48hr cooling-off |
| CRIMSON | > $50,000 | Board-level approval + Reg-Watch check |

---

## SafetyNet First

**Never skip these on any execution task:**

1. Check risk level — if RED or above, require explicit user approval
2. Append legal disclaimers on all Tax and Legal outputs
3. Log every decision to `AgentDecisionLog` with confidence scores
4. Trigger Watchdog Agent on any threshold breach
5. Never execute entity formation, banking setup, or capital transfers without user "Go-Ahead"
