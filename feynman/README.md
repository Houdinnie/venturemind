# VentureMind on Feynman — Research Agent Package

> 4 specialist agents × 4 research skills × recurring regulatory watch = continuous jurisdiction intelligence for Nomad Flow.

---

## What Is This

Feynman is the **research engine** for VentureMind — the other three frameworks handle the execution plane (Paperclip), the engineering sprint (gstack), and the evaluation harness (agent-skills-eval). Feynman handles the intelligence layer.

## The 4 Feynman Agents

| Agent | Role |
|-------|------|
| **Researcher** | Gathers evidence across legal, tax, entity, market domains |
| **Reviewer** | Audits claims against primary sources, assigns confidence scores |
| **Writer** | Converts audit reports to founder-friendly briefs |
| **Verifier** | Confirms citations resolve, dead links cleared — no document reaches founder unless CLEARED |

## The 4 Research Skills

| Skill | Purpose |
|-------|---------|
| `venturemind-deepresearch` | Multi-agent investigation for complex founder queries |
| `venturemind-lit` | Literature review for building the jurisdiction knowledge base |
| `venturemind-watch` | Recurring regulatory monitoring across 20+ jurisdictions |
| `venturemind-recipe` | Generates Green Button execution sequences from Discovery profiles |

## How It Fits

```
Founder Query
     ↓
feynman deepresearch "structuring question"
     ↓
Researcher → 4 domain sweeps
     ↓
Reviewer → confidence scoring + HITL check
     ↓
[CRIMSON/RED] → Watchdog alerted → execution blocked
     ↓
Writer → founder brief
     ↓
Verifier → citation audit
     ↓
CLEARED → delivered to founder
```
